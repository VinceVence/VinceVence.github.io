# Guwa ImageGen Evaluation

Six marketing-only illustrations were generated with the built-in ImageGen tool
using three to five production Guwa references per concept. Sources are archived
under `assets/guwa-site/source/`; transparent masters and responsive derivatives
are under `assets/guwa-site/optimized/`.

The source generations used a removable green background. The standard Guwa
palette was the only requested subject palette; chroma removal produced clean
transparent edges without changing production assets. Slight antialiasing and
tonal variation are derivatives of the approved colors, not a new palette.
