# Current Guwa Product Inventory

Classification: **A** production-ready, **B** implemented with limitations,
**C** planned, **D** removed. Website copy may describe A directly and B only
with the limitation shown here.

| Product area | Class | Verified implementation and limitation |
| --- | --- | --- |
| Onboarding | A | Welcome and multi-step activity guide with skip/next flow |
| Home | A | Daily activity, quick navigation, catalog, and optional monthly Big Quest |
| Activity discovery/catalog | A | Category browsing with duration, difficulty, and place metadata |
| Featured/daily activity | A | Curated daily activity displayed on Home |
| Reroll | A | Limited rerolls are visible on the active daily activity |
| Lock | A | Active activity can be locked for later |
| Completion | A | Completion creates a trail entry and can collect optional memory details |
| Notes and moods | A | Optional completion note and selectable mood values |
| Local photo memories | B | Guwa Pro feature; camera/gallery files and paths stay local |
| Calendar | A | Monthly Trail calendar with rest-day handling |
| Timeline | A | Completed activity history view |
| Statistics | A | Logged count, favorite type, and mood breakdown when history exists |
| Badges | A | Persistent milestone collection with existing-user-compatible progress |
| Streaks and goals | A | Derived from completion dates and displayed in Trail/Profile |
| Custom activities | B | Free creation/storage limits; Pro removes limits and adds richer options |
| Monthly Big Quest | B | Optional larger activity; once active it remains stable for the month |
| Guwa Pro | B | Entitlement works through RevenueCat; purchases require a Google Play build |
| Google Sign-In | B | Optional Firebase authentication; requires valid production configuration |
| Account Backup | B | Eligible signed-in users can back up progress; photos are excluded |
| Restore purchases | B | Implemented but unavailable outside a licensed Google Play billing build |
| Founder Pass | B | Requires sign-in, App Check, eligibility, capacity, and RevenueCat confirmation |
| Profile | A | Plan, progress, backup, Pro, promo, support, privacy, and version surfaces |
| Settings | B | Relevant settings/actions live in Profile; no separate settings tab |
| Privacy/support/deletion links | A | Centralized `guwa.app` and `support@guwa.app` targets |
| Preference/“What fits today?” flow | D | Removed from Home |
| Quest pace setting | D | Removed; Guwa does not require a daily completion |
| Public social feed | D | Intentionally absent; memories remain private/local-first |

No category C feature is advertised on the redesigned website.
